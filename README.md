# flowsmith

一系列用 AI 打造的小型工作流（AI workflow）工具集合。每個子資料夾是一個獨立、可單獨執行的小工具，主題都是「把一段固定的需求，變成可以重複輸入資料就能跑的小程式」。

## 設計理念

每個工具都遵循同一套骨架：

```
輸入資料（介面層） → 組成 Prompt（邏輯層） → 呼叫 AI（API層） → 顯示結果
```

差異只在「Prompt 模板怎麼寫」，其他結構幾乎共用，方便快速擴充新工具。

## 工具列表

| 工具 | 說明 | 路徑 |
|---|---|---|
| prepsmith | 輸入公司與職缺資訊，自動生成面試分析（公司總結、核心技能、可能問題與回答方向） | [/prepsmith](./prepsmith) |

## 技術棧

- 前端：HTML / CSS / JavaScript（純原生，無框架）
- AI：Anthropic Claude API

## 使用方式

每個子資料夾內都有自己的 `README.md`，說明該工具的具體用法。直接開啟對應的 `.html` 檔即可在瀏覽器中執行。
