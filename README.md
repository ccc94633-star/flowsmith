# FlowSmith

FlowSmith 是我的 AI 小工具作品總集。

這個 repository 會用來收集我製作的各種 AI workflow、AI 輔助工具、教學範例與實驗作品。每個子資料夾都是一個獨立作品，未來可以持續加入新的工具。

## 作品總集定位

FlowSmith 不是單一工具，而是一個 AI tools portfolio。每個作品會盡量呈現完整流程：

```text
使用者輸入 -> Prompt 設計 -> AI API 串接 -> 結果整理 -> 可使用的介面
```

我希望這個總集可以展示：

- 我如何把需求拆成可執行的 AI workflow
- 我如何設計 Prompt 與資料輸入流程
- 我如何把 AI 回應整理成可讀、可用的介面
- 我如何逐步把小工具做成可展示的作品
